# 🧭 Compass Day Bot

**Compass Day Bot** — это автоматизированная система для ежедневной рассылки персонализированного контента пользователям в Telegram. Контент генерируется с помощью ИИ (моделей Gemini или OpenRouter) и включает в себя советы от трех «экспертов»: Психолога, Стилиста и Нутрициолога.

В состав системы входит панель управления на базе **Streamlit**, где администратор может мониторить логи, управлять системными настройками (выбор ИИ-провайдера, модели, времени рассылки) и при необходимости запускать рассылку вручную.

## ✨ Основной функционал

*   **Тройная ИИ-генерация**: Ежедневные советы от психолога, стилиста и нутрициолога, сгенерированные на основе профиля пользователя.
*   **Панель управления (Streamlit)**: Удобный веб-интерфейс для просмотра аналитики, статуса систем (БД, Telegram, AI API) и истории рассылок.
*   **Динамические настройки**: Возможность переключаться между провайдерами (Gemini / OpenRouter) и менять модель ИИ прямо из веб-интерфейса без перезагрузки системы. Настройки сохраняются в базе данных.
*   **Telegram-интеграция**: Сгенерированный контент форматируется в HTML и отправляется пользователям через Telegram-бота.
*   **Автоматизация (Cron)**: Поддержка фонового запуска по расписанию (например, через GitHub Actions).

## 🗂 Структура проекта

*   `streamlit_app.py` — Главный файл веб-интерфейса Streamlit (Панель управления).
*   `ai_service.py` — Логика взаимодействия с API ИИ (Gemini, OpenRouter) и генерация контента.
*   `telegram_service.py` — Логика отправки сообщений в Telegram и проверки статуса бота.
*   `database.py` — Взаимодействие с базой данных PostgreSQL (пользователи, логи, настройки системы).
*   `cron_daily.py` — Скрипт для пакетного автоматического запуска рассылки (используется для планировщиков и GitHub Actions).
*   `prompts.py` / `prompts.txt` — Системные промпты для настройки поведения ролей ИИ.
*   `deployment_guide.md` — Инструкция по деплою (Streamlit Cloud & GitHub Actions).

## 🛠 Установка и запуск (Локально)

### 1. Клонирование репозитория
```bash
git clone <URL_вашего_репозитория>
cd "Compass Day bot"
```

### 2. Установка зависимостей
Рекомендуется использовать виртуальное окружение:
```bash
python -m venv venv
source venv/bin/activate  # Для Mac/Linux
# venv\Scripts\activate   # Для Windows
pip install -r requirements.txt
```

### 3. Настройка переменных окружения
Создайте файл `.env` в корне проекта (рядом с кодом) и добавьте необходимые доступы:

```env
DB_HOST=ваш_хост_бд
DB_PORT=5432
DB_USER=пользователь_бд
DB_PASSWORD=пароль
DB_NAME=имя_бд

GEMINI_API_KEY=ваш_ключ_gemini
OPENROUTER_API_KEY=ваш_ключ_openrouter # Опционально

TELEGRAM_BOT_TOKEN=токен_вашего_бота
```

### 4. Запуск панели управления
```bash
streamlit run streamlit_app.py
```
После запуска приложения откроется вкладка в браузере (обычно по адресу `http://localhost:8501`).

### 5. Ручной тестовый запуск скрипта рассылки (Cron)
Если вы хотите просто протестировать скрипт рассылки вне Streamlit:
```bash
python cron_daily.py
```

## 🚀 Деплой в продакшен
Проект адаптирован для деплоя на платформе **Streamlit Community Cloud**, а запуск фоновой рассылки реализован посредством **GitHub Actions**.

Подробные шаги развертывания смотрите в файле: **[deployment_guide.md](deployment_guide.md)**.
